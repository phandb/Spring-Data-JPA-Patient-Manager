The Patient Manager project is created to monitor patient's information such as medications, physicians, pharmacies, and personal info. This project is written using Spring Boot to automatically configure dependencies, properties, and beans that exist in the application. More advanced Spring Data JPA is also applied to connect database. The application is protected by Spring Security. Thymeleaf and Bootstrap are also involved to help in elegant design.

After successful login, user can perform CRUD tasks on the manager. There are four majors entities including Patient, Medication, Physician, and Pharmacy. User is able to create, read, update, and delete on each model.
